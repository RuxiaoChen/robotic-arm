  • Paper

    May, 2023

    Look-Ahead Task Offloading for Multi-User Mobile Augmented Reality in Edge-Cloud Computing

    - A MAR service-oriented task offloading scheme is designed and evaluated in edge-cloud computing networks, which greatly improved the long-term quality of service for mobile AR.
    - A more specific and refined MAR application model is established using 3 types of directed acyclic graphs (DAG) according to the MAR task interdependency.
    - A long-sighted scheduling scheme based on a modified Monte Carlo tree search is formulated, which is able to make offloading decisions with consideration of migration overhead.
